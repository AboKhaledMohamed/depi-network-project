Network Topology Details

Egypt Headquarters (HQ1)

  • Core Router: AR6 series with redundant links
  • Backup Router: AR7 for failover
  • Core Switch: S5700 L3 with VRRP
  • Access Switches: Multiple S2700 L2 switches
  • Branches: Branch 1 HQ1, Branch 2 HQ1
  • Firewall: USG6000 series

Saudi Arabia Headquarters (HQ2)

  • Core Router: AR6 series with redundant links
  • Backup Router: AR4 for failover
  • Core Switch: S5700 L3 with VRRP
  • Access Switches: Multiple S2700 L2 switches
  • Branches: Branch 1 HQ2, Branch 2 HQ2
  • Firewall: USG6000 series
